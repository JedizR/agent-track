## Delegation Rules

**Always delegate to Gemini agents:**
- SQLite schema, CREATE TABLE SQL → Backend Engineer
- CRUD implementations → Backend Engineer
- Typer command definitions → Frontend Engineer
- Rich table/board rendering → Frontend Engineer
- pytest files for completed code → QA Engineer
- pyproject.toml, GitHub Actions → DevOps Engineer
- Schema architecture decisions → Business Consultant

**Always handle yourself (Claude):**
- System design and architecture decisions
- API contract authoring (write `api-contracts/*.md` before dispatching)
- Cross-domain integration (wiring CLI → CRUD → DB)
- Final review of all Gemini outputs before committing
- Resolution of all `[ESCALATE TO CLAUDE]` flags

**Dispatch guidelines:**
- **Parallel:** Backend (schema) + DevOps (pyproject.toml) have no shared files
- **Sequential:** Frontend CLI commands depend on Backend CRUD being complete
- **Research first:** dispatch Business Consultant before finalising schema design
- **Parallel integration rule:** verify ALL `agent-output/` files exist before integrating
